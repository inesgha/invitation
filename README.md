# Luxury Wedding Invitation

A high-end cinematic wedding invitation web app built with React, Vite, Tailwind CSS, Framer Motion, SVG, and Canvas API.

Run with npm install, then npm run dev.

Customize the couple, date, venue, and copy in src/App.jsx. The ambient music file is generated locally in src/assets/music/romantic-ambient.wav and can be replaced with your licensed track.
